## 摘要
基于去噪的模型（包括扩散模型和流匹配模型）在图生成领域取得了重大进展。尽管取得了这些进步，此类模型仍受限于两个基本局限性：计算成本随节点数量呈平方级增长，以及生成过程中需要大量的函数评估。在这项工作中，我们提出了一种新型层级化生成框架，该框架减少了必须评估的节点对数量，并采用离散流匹配来显著减少去噪迭代次数。实验证明，我们的方法能更有效地捕捉图分布，同时大幅缩短生成时间。

## Abstract
Denoising-based models, including diffusion and flow matching, have led to substantial advances in graph generation. Despite this progress, such models remain constrained by two fundamental limitations: a computational cost that scales quadratically with the number of nodes and a large number of function evaluations required during generation. In this work, we introduce a novel hierarchical generative framework that reduces the number of node pairs that must be evaluated and adopts discrete flow matching to significantly decrease the number of denoising iterations. We empirically demonstrate that our approach more effectively captures graph distributions while substantially reducing generation time.

#### 1. 核心问题与研究动机
当前基于去噪的图生成模型（如扩散模型和流匹配模型）面临两个主要的计算瓶颈：
*   **计算复杂度高：** 绝大多数模型需要在所有节点对（$O(n^2)$）上运行神经网络，导致在大规模图上难以扩展。
*   **生成效率低：** 生成过程通常需要大量的函数评估次数（NFE），即多次迭代去噪，导致推理速度缓慢。
本文旨在通过引入**层级化（Hierarchical）**结构和**离散流匹配（Discrete Flow Matching）**技术，同时解决计算规模和生成速度的问题。

#### 2. 方法论核心思想
论文提出了一种“由粗到细”的层级生成框架，核心步骤如下：
*   **D-Min 聚类（核心创新）：** 提出了一种专门的图粗化算法。不同于传统的社区发现（旨在最大化模块度），D-Min 通过 GNN 学习节点分配矩阵，目标是**最小化扩展图（Spanning Supergraph）的密度**。这确保了在细化阶段只需处理极少数的候选边，显著降低了计算量。
*   **确定性扩展与细化：** 
    *   **扩展：** 将粗图的每个父节点分裂为多个子节点，父节点间的边转化为子节点间的完全二分图（Biclique）。
    *   **细化：** 使用离散流匹配模型在扩展图的稀疏结构上进行“剪枝”和属性生成。
*   **离散流匹配 (DFM)：** 采用了 Gat 等人 (2024) 的简化公式，定义了从噪声分布到数据分布的概率路径。通过插值公式和概率速度（Probability Velocity）建模，实现了在极少迭代次数（如 32 次）下高质量生成。
*   **等变性保证：** 证明了粗化操作具有置换不变性，而扩展操作具有置换等变性，确保了模型对节点顺序的不敏感。
